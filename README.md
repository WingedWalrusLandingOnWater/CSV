# CSV Repository

A collection of simple, generic CSV lists organized by topic for use by other applications and people.

## Folder Structure

- **Games/** - Game-related lists (titles, genres, studios, etc.)
- **Books/** - Book-related lists (titles, authors, genres, publishers, etc.)
- **PopCulture/** - Popular culture lists (movies, TV shows, celebrities, music, etc.)
- **AdultTheme/** - Adult-themed lists (content appropriate for adult audiences)
- **Misc/** - Miscellaneous lists that don't fit other categories

## Usage

Each subfolder contains CSV files ready for import into applications or data processing workflows. Feel free to use and contribute!

## Guidelines

- Keep CSVs simple and well-formatted
- Use clear, descriptive filenames
- Include headers in the first row when applicable
